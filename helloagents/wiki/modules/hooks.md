# hooks

## 目的
提供可复用的 Hooks 聚合与封装，提升 UI 层开发效率与一致性。

## 模块概述
- **职责:** 统一导出常用 hooks（如节流/防抖/本地存储等）
- **状态:** 🚧开发中
- **最后更新:** 2025-12-24

## 规范

### 需求: Hooks 统一出口
**模块:** hooks
常用 hooks 通过 `src/hooks/index.ts` 统一导出，减少深层路径依赖。

#### 场景: 引用 hooks
业务组件从统一入口引用 hooks，避免重复安装与分散引用。
- 导出命名稳定
- 文档/示例与导出保持一致
